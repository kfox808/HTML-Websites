Meet the Artist

As an artist, I create art that can be worn, and displayed. As a programmer, I enjoy creating websites, creative programs, and working with numbers. As a creative programmer, I enjoy creating bold masterpieces that incorporate sacred geometry and soul. I started my journey with creative coding when I studied web browser languages (HTML, CSS, JavaScript) and was introduced to the P5.js library, which is a site that allows users to create unique artwork with code. Using this library has not only been an extension of my creative expression, but it has sharpened my skills in computational mathematics. As for pixel art, I used a simple website called pixelartmaker.com. It features a grid with a full spectrum of color options to select. There, I was able to create so much.
